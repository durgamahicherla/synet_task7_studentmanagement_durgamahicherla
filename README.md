# 🎓 Student Management System

A command-line student records manager built with Python.

## 📌 Project Info
- **Internship:** Synent Technologies – Python Development
- **Task:** Task 7 – Student Management System (Intermediate Level)

## 🚀 How to Run

```bash
python student_management.py
```

> No extra libraries needed! Uses only built-in Python modules.

## 🎯 Features
- ➕ Add new students
- 📋 View all students
- 🔍 Search student by name
- ✏️ Update student details
- 🗑️ Delete student
- 💾 Data saved in CSV file (students.csv)
- ✅ Input validation
- 🔢 Auto ID generation

## 🛠️ Tech Used
- Python 3.x
- `csv` module (built-in)
- `os` module (built-in)

## 📁 Files
| File | Description |
|------|-------------|
| `student_management.py` | Main program |
| `students.csv` | Auto-created data file |
